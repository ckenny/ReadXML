Example String with XML to JSON

1. Convert string with XML content to JSON string.

JavaScript Code

<script type="text/javascript" src="xml2json.js"></script>
<script type="text/javascript">
// <![CDATA[
// string with XML
var xmlstr = '<?xml version="1.0" encoding="UTF-8"?><websites>'+
'<site id="1" pr="5"><title>Web Programming Courses</title><url>http://coursesweb.net/</url></site>'+
'<site id="2" pr="4"><title>Courses Games Anime</title><url>http://www.marplo.net/</url></site>'+
'</websites>';

// gets the JSON string
var jsonstr = xml2json.fromStr(xmlstr, 'string');

// adds the JSON string in HTML element with id="jsnstr"
document.getElementById('jsnstr').innerHTML = jsonstr;
// ]]>
</script>

Result this JSON string




2. Convert string with XML content to JSON object. Gets and adds into an HTML element the value of an attribute, and the text from a specified XML tag.

JavaScript Code


<script type="text/javascript" src="xml2json.js"></script>
<script type="text/javascript">
// <![CDATA[
// string with XML
var xmlstr = '<?xml version="1.0" encoding="UTF-8"?><websites>'+
'<site id="1" pr="5"><title>Web Programming Courses</title><url>http://coursesweb.net/</url></site>'+
'<site id="2" pr="4"><title>Courses Games Anime</title><url>http://www.marplo.net/</url></site>'+
'</websites>';

// gets the JSON object
var jsonobj = xml2json.fromStr(xmlstr);

// gets the attribute value from:  websites.site[1]['@attributes'].id
// and the text from:  websites.site[1].url['#text']
var attrid = jsonobj.websites.site[1]['@attributes'].id;
var txturl = jsonobj.websites.site[1].url['#text'];

// adds the values in HTML element with id="url2"
document.getElementById('url2').innerHTML = 'ID: '+ attrid +'<br/>URL: '+ txturl;
// ]]>
</script>



Results:






CoursesWeb.net
